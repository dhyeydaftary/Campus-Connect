# Security Policy

## Supported Versions

Currently, only the latest version of Campus Connect on the `main` branch is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| `main`  | :white_check_mark: |
| `< 1.0` | :x:                |

## Reporting a Vulnerability

We take the security of Campus Connect very seriously.

If you discover a security vulnerability, please do **NOT** open a public issue.

Instead, please report it via one of the following methods:
1. Email the project maintainers directly using the email listed in the repository documentation.
2. Use the in-app "Report an Issue" feature and select "Security" as the category, which routes directly to the platform administrators.

**Please include the following information in your report:**
* A detailed description of the vulnerability.
* Step-by-step instructions on how to reproduce the issue.
* The potential impact of the vulnerability.
* If possible, a suggested mitigation or fix.

We will acknowledge receipt of your vulnerability report as soon as possible and strive to send you regular updates about our progress fixing the issue.
