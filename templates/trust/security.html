Tips

Security Best Practices

Use Unique Passwords

Never reuse the password you use for your primary email, banking, or other critical services. Using a password manager to generate and store a 16+ character random password is highly recommended.

Public Devices & Networks

Always log out when using library computers, lab workstations, or borrowed devices. Avoid sensitive actions over unsecured public Wi-Fi without a VPN.

Beware of Phishing

Campus Connect administration will NEVER email you asking for your password. If you receive a suspicious email, do not click the links. Report it immediately.

Found a Vulnerability?

We take all reports regarding security seriously. If you've discovered a security flaw, please let us know immediately.
